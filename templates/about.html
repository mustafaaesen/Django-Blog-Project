Django Tabanlı Dinamik Blog Platformu

Bu proje, Django framework kullanılarak geliştirilmiş tam fonksiyonel bir blog platformudur. Amaç; kullanıcıların kendi profillerini oluşturabildiği, makale yayımlayıp düzenleyebildiği, yorum ve beğeni etkileşimlerinde bulunabildiği bir sistem inşa etmektir. Geliştirme sürecinde kullanıcı deneyimi, veri bütünlüğü ve ölçeklenebilir mimari ön planda tutulmuştur.

Teknolojik Altyapı

  • Backend: Python – Django Framework
  • Database: SQLite
  • Frontend: Frontend: HTML5, CSS3, Bootstrap
  • Authentication: Django User Model, Login/Register sistemleri
  • Media Management: Dinamik dosya yükleme, profil ve makale görselleri (FileField)
  • Pagination: Django Paginator sınıfı ile sayfa bazlı içerik yönetimi
  • Dynamic Content: Jinja2 tabanlı şablon yapısı, context verisi üzerinden dinamik render
  • Security: CSRF koruması, güvenli dosya yükleme, kullanıcı tabanlı erişim kısıtlaması
  • Extra Features: Makale beğeni sistemi, yorum beğenisi, kullanıcı istatistikleri, “en popüler yazarlar” analizi
